# 月別歩数アプリ

参加者の月別歩数データを可視化するWebアプリケーションです。

## 機能

- 📊 **テーブル表示**: 参加者の月別歩数を表形式で表示
- 📈 **グラフ表示**: Chart.jsを使用したインタラクティブな折れ線グラフ
- 🔍 **参加者フィルター**: 特定の参加者のデータのみを表示
- 📊 **統計情報**: 各参加者の合計、平均、最大、最小歩数を表示

## 使い方

1. `index.html`をブラウザで開く
2. テーブル表示とグラフ表示を切り替え可能
3. 参加者フィルターで特定の参加者を選択可能
4. 統計情報セクションで各参加者の詳細を確認可能

## ファイル構成

```
.
├── index.html          # メインのHTMLファイル
├── styles.css          # スタイルシート
├── app.js              # アプリケーションロジック
└── README.md           # このファイル
```

## GitHub Pagesでの公開方法

1. GitHubにリポジトリを作成
2. このファイルをリポジトリにアップロード
3. リポジトリのSettings > Pagesで、Sourceを「main」ブランチに設定
4. 数分後に `https://[ユーザー名].github.io/[リポジトリ名]` でアクセス可能
