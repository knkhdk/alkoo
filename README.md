# 日別・月別歩数アプリ

参加者の日別・月別歩数データを可視化するWebアプリケーションです。

## 機能

- 📊 **テーブル表示**: 参加者の日別・月別歩数を表形式で表示
- 📈 **グラフ表示**: Chart.jsを使用したインタラクティブな折れ線グラフ
- 🔍 **参加者フィルター**: 特定の参加者のデータのみを表示
- 📊 **統計情報**: 各参加者の合計、平均、最大、最小歩数を表示
- 👥 **チーム比較**: うさぎさんチームとかめさんチームの平均値を比較
- 🏆 **ランキング表示**: 最新月の平均歩数に基づくランキング表示

## セットアップ

### 1. 依存パッケージのインストール

```bash
npm install
```

### 2. Excelファイルからデータを生成

プロジェクトフォルダ内の`step_data.xlsx.xlsx`ファイルを更新した後、以下のコマンドを実行して`data.js`を生成します：

```bash
npm run build
```

または：

```bash
node build-data.js
```

これにより、`data.js`ファイルが自動生成されます。

### 3. ブラウザで開く

`index.html`をブラウザで開いてください。

## データの更新方法

1. `step_data.xlsx.xlsx`ファイルを編集
   - 月別データは「DATA」シート（または最初のシート）に記入
   - 日別データは「11月」「12月」などのシートに記入
   - 新しい月のデータを追加する場合は、既存の月のシートをコピーして「12月」などの名前に変更

2. `npm run build`を実行して`data.js`を再生成

3. ブラウザで`index.html`をリロード

## ファイル構成

```
.
├── index.html              # メインのHTMLファイル
├── styles.css              # スタイルシート
├── app.js                  # アプリケーションロジック
├── data.js                 # 自動生成されるデータファイル（gitに含める）
├── build-data.js           # Excelファイルからdata.jsを生成するスクリプト
├── package.json            # Node.js依存関係
├── step_data.xlsx.xlsx     # 元のExcelデータファイル
└── README.md               # このファイル
```

## GitHub Pagesでの公開方法

1. GitHubにリポジトリを作成
2. このファイルをリポジトリにアップロード
3. リポジトリのSettings > Pagesで、Sourceを「main」ブランチに設定
4. 数分後に `https://[ユーザー名].github.io/[リポジトリ名]` でアクセス可能

**注意**: Excelファイルを更新した後は、`npm run build`を実行して`data.js`を再生成し、GitHubにプッシュしてください。

## セキュリティ

- Web上でのExcelファイルの直接読み込みは無効化されています
- データは`build-data.js`で生成された`data.js`から読み込まれます
- 不正なデータの混入を防ぐため、Excelファイルはプロジェクト管理者のみが編集できます
